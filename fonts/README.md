# Fonts Folder

Place project-approved font files here for runtime caption rendering.

Recommended file types:

- `.ttf`
- `.otf`

Recommended usage:

- add Thai-capable fonts such as `THSarabun`, `TH Chakra Petch`, or another approved family
- keep the final approved font set consistent across operator machines when possible
- use clear file names so later font-resolution settings are easy to trace
- keep `captions.toml font_family` close to the actual font file stem when practical so workspace-font matching stays deterministic
